# Themes for Terminal elementaryOS
Script for change Theme Color on Terminal

This run on elementary OS 5.1 Hera

## How to Install

* Download theme.sh

* Name it as 'theme'

* Place it in /usr/bin/

```
sudo mv theme /usr/bin/
```

* Execute on your Shell_prompt

```
$ theme
```
